# Installing mckinsey-research Skill

## Supported Platforms

This skill works with any AI agent that supports SKILL.md-based skills:

| Platform | Install Path | Command |
|----------|-------------|---------|
| **OpenClaw** | `~/.openclaw/skills/` | `clawhub install mckinsey-research` |
| **Antigravity** | `agent/skills/` | Copy folder to `agent/skills/mckinsey-research/` |
| **Augment** | `~/.augment/skills/` | Copy folder to `~/.augment/skills/mckinsey-research/` |
| **Claude Code** | `~/.claude/skills/` | Copy folder to `~/.claude/skills/mckinsey-research/` |
| **Cline** | `cline/skills/` | Copy folder to `cline/skills/mckinsey-research/` |
| **CodeBuddy** | `~/.codebuddy/skills/` | Copy folder to `~/.codebuddy/skills/mckinsey-research/` |
| **Command Code** | `~/.commandcode/skills/` | Copy folder to `~/.commandcode/skills/mckinsey-research/` |
| **Codex** | `agent/skills/` | Copy folder to `agent/skills/mckinsey-research/` |

## Quick Install (Any Platform)

1. Download or clone the `mckinsey-research` folder
2. Copy it to your agent's skills directory
3. The skill auto-triggers when you ask for market research or strategy analysis

## Via ClawHub

```bash
npm i -g clawhub
clawhub install mckinsey-research
```
